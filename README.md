# TaskFlow 

A modern Jira-inspired Project Management Tool built using HTML, CSS, JavaScript, Chart.js, jsPDF, Drag & Drop API, and LocalStorage.

TaskFlow Pro helps users manage projects, organize tasks, track productivity, monitor deadlines, and analyze team performance through a modern dashboard interface.

---

##  Features

###  Project Management
- Create and manage projects
- Set project deadlines
- Assign priorities
- Project overview dashboard

###  Task Management
- Create tasks
- Edit tasks
- Delete tasks
- Favorite important tasks
- Assign tasks to team members
- Due date tracking
- Task priorities

###  Kanban Board
- Drag & Drop functionality
- To Do column
- In Progress column
- Review column
- Done column
- Automatic status updates

###  Team Management
- Add team members
- Assign tasks
- Team overview
- Productivity tracking

###  Analytics Dashboard
- Task status chart
- Priority distribution chart
- Completion percentage
- Productivity insights

###  Calendar View
- Upcoming deadlines
- Due task tracking
- Overdue task detection
- Deadline monitoring

###  Productivity Features
- Daily goal tracking
- Productivity streak
- Activity logs
- Smart notifications

###  Export Features
- Export PDF reports
- Export CSV reports
- Analytics reports

###  Data Management
- LocalStorage persistence
- Backup data (JSON)
- Restore backup
- Delete all data

###  User Interface
- Responsive design
- Dark mode
- Toast notifications
- Modern SaaS dashboard UI
